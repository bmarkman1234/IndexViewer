IndexViewer

A lightweight web app for calculating and visualizing indices from airborne and spaceborne imagery

Workflow

Step 1

Upload Bands

Upload one or more raster bands (`.tif/.tiff/.png/.jpg`), then assign a clear alias to each band.

    Formula Help (Aliases, Helpers, Constants)

    Rename each uploaded band with an alias, then use alias names in formulas. Use alias names directly (for example, `B4`) or `[raw file name]` for direct references.

    Need index ideas? Browse formulas at IndexDatabase.

    Step 4

    Save And Compare

    Each calculation is saved using its index label so you can revisit and compare results.
