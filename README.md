# Index Viewer

Simple browser app for viewing and analyzing satellite index rasters.

## Basic Functionality

- Upload raster bands (`.tif/.tiff/.png/.jpg`)
- Rename bands with aliases
- Enter a raster algebra formula to calculate an index
- View the calculated index on canvas and map overlay
- Save multiple index snapshots over time
- Use the change slider to compare snapshots
- Click the map to inspect pixel values and trend over time
- Export index image as `.png`
- Export pixel probe data as `.csv`
